「Webコンテンツ作成」の成果

本テーマについて

先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得することが目的です。

到達目標は次の4つです

  1. 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できる
  2. コミュニケーション能力の重要性を理解し、グループワークを円滑に実施できる
  3. モーション解析・編集ができる
  4. HTML等を用いたWebページ作成ができる

なお、本授業は、リーダーシップ能力、課題に対する解決能力育成に寄与する科目です。

プロジェクト課題について

今回私が選んだのは「グルメ情報」です。genreのとりうる値をラジオボタンで指定するラジオボタンを設置して、検索ボタンを押すことで最新の検索結果が反映されます。 個人的に一番大変だったのが、ajax通信を利用するときにどのデータの部分から取り出せばいいのかがわかりにくかったところです。 また、検索した最新のものが後ろに来ていたので前のデータをどう消せばいいのかをhtmlの要素を見直して訂正する必要がありました。 そのほかの作業は特に問題はなかったです。

プロジェクト課題で作成したページ:project.html

数当てゲーム

このゲームは1から10までの整数の中で、コンピュータがどの数字を考えているかを当てるゲームです。 チャンスは3回あり、3回目の回答ボタンを押してから解答が表示されゲームは終了します。 その後回答ボタンを何度も押しても'答えは 〇〇 でした．すでにゲームは終わっています'と表示されます。 なお、ヒントとして'まちがい．答えはもっと　大きい/小さい　ですよ'と表示するように書いてあるので、 回答者はそれを頼りに回答をすることができます。 授業内ではjs、dom、イベント処理の順番でこのゲームを段階ごとに作成することができました。 特にdomは初めて使う関数がかなり多く、その名前は長く見分けがつきにくかったのでかなり時間がかかりました。

数当てゲームのページ:kazuate.html

各授業回の学習事項

第1回 WWW と HTML

webページとhtmlの概略や、それらを使うと私たちは何ができるのかについて学びました。 webページの部品として、他にもcss、javascript、画像、音声、動画が含まれています。 htmlではp要素やa要素、strong要素などの要素を用いることでwebページの基礎を作ることができます。

演習の成果: 演習1-4: HTMLの様々な要素

第2回 CSS による Web ページの装飾

cssとは、htmlを元にwebページの文字のフォントや色、背景、アクセサリーのようなものをつけることが可能になる部品です。 cssをいじることで各々の個性がわかりやすく反映され、webページがあたかも面白く見えるのが利点です。

演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

JavaScriptはWebページ上で複雑な機能を実行することが可能です。 Webページを開いた直後やボタンを押したときにプログラムが実行され、動きのあるものに作り替えることができます。 多くのブラウザはjavaScriptを通してAPIを提供してくれます。

演習の成果: 練習3-6: オブジェクトの参照

第4回 DOM と JavaScript

DOM（Document Object Model）はWebコンテンツを表現するデータ構造で、 jsなどを用いることでコンソールに結果を表示したり変更を加えたりする処理が書けるようになります。 ajax通信の結果をdomに事前にどの部分を出すかを書いておくことで、見たい結果を見ることができます。

演習の成果: 練習4-2~4-5: DOMプログラミング

第5回 イベント処理

イベントとは、Webページ上で起こった動作や出来事のことで、その処理について学びました。 カーソルを移動方法だけで様々な処理があることから、cssのように個性が溢れる部分です。 function 〇〇(←関数名)(){}を記述することで、操作が可能です。

演習の成果: 練習5-3: イベントを選択する
